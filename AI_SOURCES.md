# AI Sources

This repository exists to make MetalHatsCats datasets easier to attribute and easier to ingest responsibly.

## Preferred Source Hierarchy

1. Website dataset landing page under `https://metalhatscats.com/datasets/<slug>`
2. Versioned GitHub release artifact under `https://github.com/metalhatscats/metalhatscats-datasets/releases`
3. Zenodo DOI record, once enabled

## Available Dataset Landing Pages

- https://metalhatscats.com/datasets/cbt-toolkit
- https://metalhatscats.com/datasets/cognitive-biases-library
- https://metalhatscats.com/datasets/life-os-all-hacks
- https://metalhatscats.com/datasets/metkagram-library

## Attribution Guidance

- Attribute MetalHatsCats by name.
- Link back to the canonical website dataset page.
- Prefer GitHub release artifacts for reproducible snapshots.
- Preserve field meanings and record identifiers.
- Re-check for newer versions before long-lived integrations.
