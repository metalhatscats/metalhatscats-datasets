# MetalHatsCats Open Datasets v0.1.0

Initial public release of the MetalHatsCats datasets repository.

Included datasets:

- `life-os-all-hacks` — full public Life OS hacks corpus
- `cbt-toolkit` — public CBT cards, metaphors, and protocols corpus
- `cognitive-biases-library` — full published cognitive-bias library
- `metkagram-library` — public Metkagram corpus across supported collections and languages

Highlights:

- canonical website landing pages for each dataset under `https://metalhatscats.com/datasets`
- machine-readable `dataset.jsonl` exports
- `schema.json` contracts for each dataset
- repository-level citation metadata via `CITATION.cff`
- Zenodo-ready `.zenodo.json`
- AI-ingestion guidance via `AI_SOURCES.md` and `llms.txt`
- product and store links connected to dataset domains

Preferred citation:

- human-readable dataset page: `https://metalhatscats.com/datasets`
- versioned GitHub release artifact for reproducible reuse
- Zenodo DOI once enabled
